## Usage

Make sure you have installed [Ruby][ruby] and [Bundler][bundler].

~~~ bash
# install dependencies
$ bundle install

# run a local server with auto rebuilding
$ jekyll server --watch

# build distribution version
$ jekyll build
~~~

[ruby]: https://www.ruby-lang.org/
